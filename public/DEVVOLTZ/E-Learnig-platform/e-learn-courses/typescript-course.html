Welcome to TypeScript

TypeScript is the most widely used programming language for web development. This tutorial will guide you through key TypeScript concepts with practical examples.

üìå What is TypeScript?

  • TypeScript is a superset of JavaScript that adds static typing and modern language features.
  • It helps catch errors at compile-time and makes large codebases easier to maintain.
  • TypeScript code is transpiled to JavaScript, so it runs anywhere JavaScript runs (browser, Node.js).
  • Works seamlessly with existing JS and libraries‚Äîgradual adoption is possible.
  • TypeScript files typically use the .ts (or .tsx) extension.

Example: Type-safe function

// add.ts
function add(a: number, b: number): number {
  return a + b;
}
// add(5, "10"); // ‚ùå Compile-time error

üí° Why Use TypeScript?

TypeScript makes your code safer, more structured, and easier to maintain. It helps you catch errors before running the program, improves code readability with types, and provides powerful tooling support like auto-completion and refactoring. TypeScript is especially valuable in large-scale applications where reliability and maintainability are important.

üß™ TypeScript Example:


// app.ts
const demoElement = document.getElementById("demo") as HTMLElement;
demoElement.innerHTML = "Hello TypeScript!";

This example shows how TypeScript uses type assertions to ensure demoElement is an HTMLElement. You can compile it with tsc app.ts which generates app.js, and then run it in your HTML with an element like <p id="demo"></p>.
