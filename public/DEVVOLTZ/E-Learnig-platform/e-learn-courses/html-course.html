Basic HTML Structure

An HTML document starts with a `<!DOCTYPE html>` declaration, followed by `<html>`, `<head>`, and `<body>` tags. Here's an example:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Page</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>This is my first web page.</p>
</body>
</html>

Try It Live

Common HTML Elements

  • <h1> to <h6>: Define headings of different levels.
  • <p>: Create paragraphs for text content.
  • <a href="...">: Add hyperlinks to other pages or resources.
  • <img src="..." alt="...">: Embed images.

Example: Adding a Link

<a href="https://example.com">Visit Example</a>
