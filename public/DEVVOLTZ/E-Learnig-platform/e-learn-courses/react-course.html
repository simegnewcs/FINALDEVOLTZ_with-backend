Welcome to React

React is a powerful JavaScript library for building user interfaces, widely used for creating single-page applications and reusable UI components. This tutorial, inspired by W3Schools, guides you through React concepts with practical examples, exercises, and quizzes to enhance your learning experience.

üìå What is React?

  • React is a JavaScript library developed by Facebook for building user interfaces.
  • It enables the creation of reusable UI components, enhancing modularity.
  • Utilizes a virtual DOM for efficient rendering and performance.
  • Supports gradual adoption, integrating seamlessly with existing JavaScript projects.
  • Typically uses .jsx or .tsx file extensions.

Example: Simple component

// Greeting.jsx
import React from 'react';
function Greeting(props) {
  return <h1>Hello, {props.name}!</h1>;
}
// <Greeting name="World" />; // Renders: Hello, World!

üí° Why Use React?

React simplifies building dynamic, component-based, and performant user interfaces. It‚Äôs ideal for single-page applications and large-scale projects due to its:

  • Component-based architecture: Reuse UI components across your app.
  • Virtual DOM: Optimizes updates for faster rendering.
  • Hooks: Simplifies state and lifecycle management in functional components.
  • Community and ecosystem: Integrates with tools like Redux, Next.js, and more.

üß™ React Example:


// App.jsx
import React from 'react';
import ReactDOM from 'react-dom/client';

function Hello(props) {
  return <h1>Hello {props.name}!</h1>;
}

const container = document.getElementById("root");
const root = ReactDOM.createRoot(container);
root.render(<Hello name="World" />);

This example demonstrates rendering a simple React component. Include it in an HTML file with a <div id="root"></div> element.
